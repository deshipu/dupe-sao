Those are design files for the printed circuit boards for the Duplicant SAO.
They have been created using [EasyEDA online editor](https://easyeda.com/editor) and can be imported using the file→open option from the menu in that program.
A PDF of the schematic is included for convenience.

The designs can be sent to fabrication at JLCPCB directly from the program, or
you can generate gerber files and send them to any other PCB fab house. I
recommend using white PCB.
